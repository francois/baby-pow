Baby Pow
========

Using the LÖVE game engine, this keyboard basher lets kids play.

Installation / Usage
====================

1. Get LÖVE:  http://love2d.org/
2. Run rake
3. Bash keyboard

WARNING: Only J and A keys supported at the moment (pre-alpha software?)
